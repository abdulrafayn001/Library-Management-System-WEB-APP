About Project



Library Management System is a Web Application that is used to manage and store the details of books and the users.In this Web Application, we have two types of users, admin, and students. Students can register and log in. After successful login, a student can see all the available books in the library. Students can also search books by their name and category. When a student takes any book from the library, the information of the book with student details and due date will automatically store in the database and update on the webpage. The student can also request a book. All the information about borrowed books, total books pending to be returned, and total fine is also shown on the student dashboard. Students can also change its password. Admins have their dashboard where they can add or remove books from the database. They can also change their password. They can also add or remove any student.

Modules


Admin Login


Admins have their separate login from where they can access the admin dashboard. There is no registration for admins. Admins can only get their login details (email and password) from the IT team.

User Register and Login


Users can be anyone who is lending books from the library. User registration is compulsory for login. After successfully login in, the user goes to the user dashboard where he/she can get all the information about the available books.

Add and Remove users


Admins can add or remove books and users from the database.

Calculate Fine


Fine Calculator is an inbuilt feature of the Library Management System. When the due date of the book crosses the system automatically calculates fines and shows them to users.

Books


Admin can add or remove from the database. Users can see all the books and search books based on name and category.

Lend Book


When a student lends a book from the library, all the information about the book with student detail and due date will automatically store in the database.

Main Page


The Main Page of this Web Application contains all the basic information about the library. It has a login and register button. Users can click on it to go to the registration or login form.
